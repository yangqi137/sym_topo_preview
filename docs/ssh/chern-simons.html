绕数与贝里相位

  • SSH模型与绕数不变量
  • 绕数与贝里相位

\(\DeclareMathOperator{\tr}{tr}\) 本章中我们计算拓扑不变量的方式看上去与上一章中存在很大的区别： 在上一章中，我们主要关注于价带子空间的性质，特别是对应的向量丛上的贝里联络。 但是在本章中，我们并没有求解哈密顿量并得到价带子空间，而是用平带哈密顿量本身就直接计算得到了绕数这一拓扑不变量。 事实上，绕数也可以从价带子空间的贝里联络和贝里相位得到。 只不过该拓扑不变量过于简单，可以直接从哈密顿量得到。 所以本章（和大部分文献）选择从哈密顿量直接计算。 在本节中，我们简要介绍绕数和能带的贝里联络、贝里相位之间的关系。

为简单起见，我们主要考虑一维的AIII类型能带模型。 事实上，一个能带的价带子空间和相应的平带哈密顿量之间存在一一对应的关系，因此利用价带子空间（或向量丛）和平带哈密顿量能得到的拓扑信息原则上是相同的。 这个对应关系也很简单：从平带哈密顿量自然可以解出其价带子空间； 从价带子空间出发，我们可以构造如下的平带哈密顿量： \[\hat H(k)=1 - 2\hat P(k).\] 其中\(\hat P\)是价带子空间的投影算符： \[\hat P(k)=\sum_{a=1}^n|\psi(k)_a\rangle\langle\psi(k)_a|,\] 这里\(a=1,\ldots,n\)表示价带子空间的一组正交归一的基底。 因此，平带哈密顿量和价带子空间是一一对应的。

具体到一个一维AIII类型能带模型，从平带哈密顿量的分块对角的形式： \[H(k)=\begin{pmatrix}0&q(k)\\q(k)^\dagger&0\end{pmatrix},\] 我们不难构造其价带子空间。 容易验证，这个平带哈密顿量的本征值为\(-1\)的本征态可以取成如下形式： \[|\psi_a(k)\rangle = \frac1{\sqrt{2}} \begin{pmatrix}|e_a\rangle\\-q(k)^\dagger|e_a\rangle\end{pmatrix},\] 其中\(|e_a\rangle\)是一组\(\mathbb C^n\)的正交归一基底。 这组基底当然可以取得非常的平庸，我们只是不愿意引入一些复杂的符号来表示这样一个简单的数学概念。 特别地，这样一组标准的基底当然可以取得与动量\(k\)无关。

将上述本征态基底代入上一章中的贝里联络的定义中，我们可以计算其对应的贝里联络。 计算过程并不复杂，这里就不赘述了。 很容易注意到，由于上述基底中只有矩阵\(q\)依赖于\(k\)，因此最后得到的贝里联络也只由\(q\)决定： \[\mathcal A = \frac i2 q(k)^{-1}\frac{\partial q(k)}{\partial k}. \](1) 与上节中绕数的公式比较，我们看到\(\tr\mathcal A\)与绕数公式中的被积函数只差一个\(\frac i2\)的系数。 因此，上一章中定义的贝里相位\(\gamma\)可以和绕数联系起来， \[\gamma = \int_0^{2\pi}\tr\mathcal A = w\pi.\] 相应地，围绕整个布里渊区的和乐矩阵也可以和绕数联系起来， \[\det\Phi = e^{i\gamma}=e^{iw\pi}.\]

这个结果很容易推广到三维：事实上，(1)可以直接推广到3维， \[\mathcal A^a = \frac i2 q(\boldsymbol k)^{-1}\frac{\partial q(\boldsymbol k)}{\partial k_a}. \](2) 因此，三维的绕数公式也可以改用贝里联络写出来。 读者可以自行尝试推导一下，这里就不再赘述了。
