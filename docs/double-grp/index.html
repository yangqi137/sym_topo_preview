费米子与双群对称性

在本章中，我们将介绍空间群对称性在实际材料体系中的重要的拓展：双空间群(double space group)。 我们通常将其简称为双群和磁群。 出现这种拓展的原因是在电子体系中，除了空间对称性之外，还存在着一种基本的对称性：费米子奇偶对称性(fermion parity symmetry)。

费米子奇偶对称性并不是一种物理对称性，它更像是我们对包含费米子的量子力学体系的描述方式中的一种数学约束条件。 因此，与其它的物理对称性不同，费米子奇偶对称性是不可能被(自发或非自发地)破缺的。 数学上，费米子奇偶对称性应该与空间群对称性等其它对称性一起，构成系统的对称群。 同时包含费米子奇偶对称性与空间群对称性的对称群就是双空间群。 由于费米子奇偶对称性操作与空间对称性操作之间并不是简单的直乘的关系，双群具有更加复杂有趣的数学结构。 同时，为了描述电子等费米子激发在晶体中的能带结构，我们应该使用双群的一类被称为双值(double value)表示的不可约表示。 我们将在第一节和第二节中详细介绍费米子奇偶对称性、双群与双值表示。
