陈绝缘体与整数量子霍尔效应

在上一章中，我们通过定义陈数这一拓扑不变量，给出了二维A对称性类型能带的拓扑分类。 由于篇幅限制（我们想让各章长度大致相当），上一章的讨论主要是从能带的数学结构出发进行的，虽然证明了具有非零陈数的能带一定是拓扑非平庸的，但是并没有给出具有非零陈数的能带的具体例子，也没有讲解非零陈数的物理意义。 在本章中，我们将通过对具体能带模型和可观测的物理量、物理效应的讨论，弥补上一章内容的缺憾，完成对二维A对称性类型能带拓扑分类的完整讨论。

具体地，我们将在第一节中给出一个具有非零陈数的例子，即二维六角晶格上的霍尔丹（Haldane）模型。 特别地，我们将看到能带陈数变化与能带结构中狄拉克点的能隙反转之间的关系。 这种计算技巧将在后续章节中被反复使用。 在后面的两节中，我们将论证具有非零陈数的二维绝缘体，即所谓“陈绝缘体（Chern Insulator）”，与整数量子霍尔效应态属于同一类拓扑物态；而能带的陈数就对应量子化的霍尔电导。 如果要严格地证明这一点，其实需要利用量子多体理论中的线性响应理论，特别是久保（Kubo）公式，来计算一个费米子体系的电导率，并从该结果出发证明霍尔电导与陈数的关系。 这种证明方式需要久保公式作为理论基础。 在本书中，我们既不愿意要求读者掌握久保公式等量子多体理论作为阅读基础；又不想推导久保公式。 因此，我们会采用另一种比较迂回，但是不涉及量子多体理论的理论推导，同时也更加直观、更加反映陈绝缘体态的拓扑本质的论证方式。 在第二节中，我们将从整数量子霍尔效应态出发，通过“劳扶林论证（Laughlin’s argument）”的推理方式，证明任意二维绝缘体系统的霍尔电导都是量子化的；并且其数值可以通过一个“电荷泵浦（charge pumping）”的理想实验过程得到。 在第三节中，我们将针对二维陈绝缘体，计算其电荷泵浦的值。我们将引入一维能带结构的“电荷极化（charge polarization）”的概念，并利用它计算二维陈绝缘体的电荷泵浦值。 利用这种方法，我们将证明二维陈绝缘体的霍尔电导正比于其能带的陈数。 因此，二维陈绝缘体就是二维整数量子霍尔效应态在格点上的实现，例如霍尔丹模型就实现了\(n=1\)的整数量子霍尔效应。
